Project Title:
NBA Simulator. 

Project Description: 
A (very crude) simulator of the NBA, allowing the user to add (and remove) teams and players, execute trades, simulate games, etc. 
When the program is run, there will be a list of available commands printed with instructions. 

Prerequisites: 
Install the latest JDK and an Java IDE of your choice. 

Usage Guide:
Clone the repository. 
Import the project to the IDE. 
Run the program.
